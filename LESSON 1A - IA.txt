<h2>Reflection</h2>

<p ondblclick="highlightAnswer()">
In this lesson, I learned about CSS text formatting properties, which control how text appears and behaves on a webpage. These properties are crucial for making content readable, visually appealing, and properly formatted across different devices.  

One key property is <span class="highlight">font-variant</span>, which can transform lowercase letters into smaller uppercase letters using the <span class="highlight">small-caps</span> value. This creates consistent capitalization while maintaining a visual hierarchy.  

Another property, <span class="highlight">text-shadow</span>, allows us to add shadows to text. By adjusting horizontal and vertical offsets, blur, and color, we can make text more visually striking or improve readability against backgrounds.  

Finally, <span class="highlight">word-wrap</span> helps control long words that exceed a container's width. Using <span class="highlight">break-word</span> ensures text stays within its boundaries, preventing layout issues and keeping the design responsive.  

Understanding these properties enhances both the style and functionality of web pages. Applying them thoughtfully allows for professional, readable, and visually attractive text.
</p>
