Lesson 4: Formatting Borders & Tables

Reflection and Example

Reflection

In this lesson, I learned about CSS borders, padding, and table formatting. Borders create visible boundaries around elements, and they can be customized by width, color, and style. Padding adds space between content and the element’s border, improving readability and layout balance. CSS allows applying padding to any element, unlike HTML padding, which mainly applies to tables. I also explored styling tables using CSS, including setting borders, cell padding, background colors, and text alignment. Properly formatting tables makes data more organized and visually appealing. Mastering these properties enhances webpage structure, readability, and overall design aesthetics.

Example Code

Hover over the code box to see which CSS properties are applied.

<style>
div {
  border: solid red 4px;
  padding: 20px;
  background-color: #ffffcc;
}
table {
  border-collapse: collapse;
  width: 80%;
}
th, td {
  border: 2px solid #333;
  padding: 12px;
  background-color: #e6f0ff;
  text-align: center;
}
</style>

<div>This div has a red border, padding, and background color.</div>

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>Course</th>
  </tr>
  <tr>
    <td>John</td>
    <td>16</td>
    <td>ICT</td>
  </tr>
  <tr>
    <td>Mary</td>
    <td>15</td>
    <td>TLE</td>
  </tr>
</table>
