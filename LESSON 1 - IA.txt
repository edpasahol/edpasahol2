<h2>Reflection</h2>

<p ondblclick="highlightAnswer()">

In this lesson, I learned about CSS typography and how it controls the appearance of text on a webpage. Typography is important in web design because it improves readability and helps organize information clearly for users. By using CSS properties, developers can customize how text looks and how it is displayed on different parts of a webpage.

One of the key things I learned is the use of <span class="highlight">font-family</span>, which allows us to choose the typeface used for text. This property can include several fonts so that if one is not available, the browser will use the next one in the list. I also learned about <span class="highlight">font-size</span> and <span class="highlight">font-weight</span>, which control the size and thickness of text. These properties help emphasize important information and make content easier to read.

Another important concept is text formatting using properties such as <span class="highlight">text-align</span>, <span class="highlight">line-height</span>, and <span class="highlight">letter-spacing</span>. These properties control how text is spaced and aligned, which improves the overall structure and visual appearance of a webpage. Proper spacing between lines and characters helps prevent text from looking crowded.

Overall, this lesson helped me understand how typography plays a major role in web design. By using different CSS typography properties, I can make webpages more readable, organized, and visually appealing. This knowledge will help me create better and more professional-looking websites in the future.

</p>
