Lesson 2: Advanced CSS Text Formatting

Reflection and Example

Reflection

In this lesson, I explored more CSS text formatting properties to enhance webpage readability and visual appeal. I learned how to bold text using font-weight and italicize text with font-style. I also discovered how to control text layout with text-indent and text-align, which help structure content neatly. Adjusting line-height improves spacing between lines for better legibility. The text-transform property lets me control capitalization using uppercase, lowercase, or capitalize options. Applying color and background styling enhances emphasis and aesthetic appeal. Overall, this lesson reinforced the importance of combining multiple CSS properties to create professional, well-formatted, and visually engaging text on webpages.

Example Code

Hover over the code box to see which CSS properties are applied.

<style>
p.example {
  font-weight: bold;
  font-style: italic;
  text-indent: 30px;
  text-align: justify;
  line-height: 1.8;
  text-transform: capitalize;
  color: #333333;
  background: #ffff99;
  font-family: 'Arial', sans-serif;
  font-size: 18px;
}
</style>

<p class="example">this is a sample paragraph demonstrating bold, italic, indented, justified, and capitalized text with line spacing, font changes, text color, and background color.</p>
